# DP-web

## The files needed to include
  - all html file
  - img folder
  - fonts folder
  - all min.js files
  - vendor folder
  - css folder

### install node packages
`npm install`

### start dev mode (hot reload)
`npm start`

### build everything
`gulp`

### compiles SCSS files into CSS and minifies the compiled CSS
`gulp css`

### minifies JS file
`gulp js`

### copy dependencies from node_modules to the vendor directory
`gulp vendor`
